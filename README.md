# fileviewflask

Uma serviço Flask para visualizar arquivos de um servidor

## Instalação

```bash
python -m venv novaPasta
cd novaPasta
source bin/activate
pip install flas

git clone git@github.com:segodimor2d2/fileviewflask.git
cd fileviewflask
python app.py
```

## Como usar

Acesse a aplicação em http://localhost:8080
